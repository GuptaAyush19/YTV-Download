# YTV-Download
Download YouTube videos on Windows using an interactive GUI

## Setup Guide
Run the following command in the terminal. This command will download all the dependencies required in your local machine. 
```
pip install -r requirements.txt
```
## Execute the application
To execute this program open command terminal and type this command this program to see the debugging statements.
```
python main.pyw
```
Else double-click `main.pyw` in project directory to execute without console.
